Welcome to the Share-a-meal API
Use the following url extensions to get the API's functionalities:
    
Users
- /api/users
- /api/users/post
- /api/users/profile
- /api/users/id
- /api/users/update
- /api/users/remove

Meals 
- /api/meals
- /api/meals/post
- /api/meals/update
- /api/meals/remove

Note that many functionalities won't work with the proper login or token
